## VISUAL DESIGN NOTES

### Color Scheme
- **Primary Blue:** #0080ff (current A&F theme)
- **Accent Colors:**
  - Customer: #33a3ff (light blue)
  - Inventory: #ff6b6b (coral/red for alerts)
  - Revenue: #51cf66 (green for growth)
  - Route: #ffd43b (yellow/gold)

### Typography
- **Headings:** Inter, 700 weight
- **Body:** Inter, 400-500 weight
- **Buttons:** Inter, 600 weight, uppercase

### Spacing
- Cards: 16px padding, 12px margin between
- Sections: 24px vertical spacing
- Content: Max-width 1200px, centered

### Icons
- Use emoji or icon library (Font Awesome, Material Icons)
- Consistent icon set for each domain

### Responsive Behavior
- Desktop: 3-column grid for cards, 4-column for buttons
- Tablet: 2-column grid
- Mobile: Single column, stacked
