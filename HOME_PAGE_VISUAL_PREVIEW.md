## INTERACTIVE ELEMENTS

### Hover States
- **Cards:** Slight elevation (shadow increase)
- **Buttons:** Color darkens, scale slightly (1.02x)
- **Example Questions:** Background highlight

### Click Actions
- **Quick Action Cards:** Expand to show more details or filter starter buttons
- **Starter Buttons:** Send the associated message to chat
- **Example Questions:** Click to use as chat input
- **Data Domain Cards:** Expand/collapse to show more info

## RESPONSIVE BEHAVIOR

### Desktop (> 1200px)
- 3-column grid for quick action cards
- 4-column grid for starter buttons
- Sidebar always visible
- Full-width content area

### Tablet (768px - 1200px)
- 2-column grid for quick action cards
- 3-column grid for starter buttons
- Sidebar collapsible
- Slightly reduced padding

### Mobile (< 768px)
- Single column layout
- Cards stack vertically
- Starter buttons: 2 columns
- Sidebar hidden (hamburger menu)
- Reduced font sizes
- Touch-friendly button sizes (min 44px height)
